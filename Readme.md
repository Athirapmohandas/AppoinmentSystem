# Appointment System

## 🚀 How to Run the Project

### 🖥️ Running the Client

1. Install dependencies:
   npm install
2. To run
   npm run dev

### 🖥️ Running the Client

1. create Virtual environment
   python -m venv env
   ./env/Scripts/activate

2. install dependencies
   pip install -r requirements.txt
3. To run
   uvicorn app.main:app
